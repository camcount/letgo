# Product Overview

Letgo is a multi-threaded security testing tool designed for penetration testing and security research. The tool provides comprehensive capabilities for:

- **Password cracking** - HTTP/HTTPS login endpoints, SSH, and hash cracking with multi-user support
- **DDoS testing** - Multiple attack modes including flood, slowloris, HTTP/2 stream flood, and RUDY attacks
- **Vulnerability scanning** - Login endpoint discovery, secret scanning, and path traversal detection
- **Proxy management** - Proxy scraping, validation, and rotation for anonymized testing

## Key Features

- Multi-threaded architecture with configurable concurrency
- Real-time progress tracking with ETA calculations
- cURL configuration import for easy attack setup
- Template-based DDoS attack configuration
- Comprehensive proxy support with health monitoring
- Cross-platform builds (Windows, Linux, macOS)

## Target Users

Security professionals, penetration testers, and researchers conducting authorized security assessments on systems they own or have explicit permission to test.

## Important Note

This tool is intended for legitimate security testing purposes only. Users must ensure they have proper authorization before testing any systems.
